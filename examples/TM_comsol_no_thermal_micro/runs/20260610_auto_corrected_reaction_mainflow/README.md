# Auto corrected reaction mainflow

This package documents the example-local implementation that closes the chain:

`training checkpoints -> exact dPi/dDelta -> corrected_stress_strain_by_step.csv -> plot_clean_tm_results.py`

Scope is restricted to:

`D:/ProgramData/PINN/FEM-PINN-main/examples/TM_comsol_no_thermal_micro`

No shared `source/` file was modified for this task.

## Files

- `REPORT.md`: main implementation and verification summary.
- `tables/corrected_reaction_by_step_seed42_smoke.csv`: exact checkpoint reaction rows generated by the new example-local postprocessor.
- `tables/corrected_stress_strain_by_step_seed42_smoke.csv`: corrected stress-strain CSV generated from those exact reactions.
- `tables/plot_auto_discovery_stress_strain_data_seed42.csv`: plotting output after auto-discovering the corrected CSV.
- `tables/mainflow_smoke_summary.csv`: compact numerical check.
- `figures/figure_summary.md`: text summary of included figures.
- `artifacts/corrected_reaction_postprocess.py`: new example-local postprocessor.
- `artifacts/main_after.py`: example-local `main.py` after the postprocess hook.
- `artifacts/plot_clean_tm_results_after.py`: plotting script after adding `result_dir/curves` auto-discovery.
- `artifacts/test_corrected_reaction_postprocess.py`: tests added for this change.
